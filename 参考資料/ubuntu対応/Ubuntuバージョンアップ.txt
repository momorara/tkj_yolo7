Ubuntuバージョンを確認

lsb_release -r

Ubuntu 24.04 LTSが最新

古いバージョンから一気にUbuntu 24.04 LTSへはバージョンアップできない。

-------------------
ultralyticsがインストールできない理由はpythonが古いからです。
Pythonのバージョンを上げる方法として3つあります。

1. pyenvをインストールして必要なpython環境を作る
  pyenvをインストールするのに、ちょっと面倒なインストール作業が必要

2. Ubuntuをステップアップバージョンアップする
　今のバージョンは結構古いので、何回かバージョンアップする必要があります。
　都度ultralyticsがインストールできるようになるまでバージョンアップする
　時間もかかるし、リスクがあります

3. 最新Ubuntuをクリーンインストールする
　最新インストーラーの入ったUSBメモリを作る必要がある
　リスクは少ない

ただし、2,3では現在のシステムバージョンが変わるので、運用中のものがあると影響が出る可能性がある。











sudo apt update
sudo apt upgrade
sudo apt dist-upgrade
sudo apt autoremove

sudo do-release-upgrade　1つ飛ばしごとに段階的にアップグレードしていく

時間がかかるし、リスクもある

クリーンインストールが良い

クリーンインストール手順
・すべてのデータを外部ドライブにバックアップします。
・Ubuntuの公式ウェブサイトからUbuntu 24.04 LTSのISOイメージをダウンロードします。
・USBメモリにブート可能なインストールメディアを作成します。
・PCを再起動し、作成したUSBから起動します。
・インストール時に「ディスクを消去してUbuntuをインストール」を選択します。
